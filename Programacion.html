Lenguaje C y C++

Algunas diferencias entre el lenguaje C y el C++

• En C, si una función no toma parámetros, tienes que añadir un voit, mientras que en C++, esto es opcional

• En C se debe declarar las variables al principio del código, mientras que en C++ puedes declararlas de forma local

• Y la gran diferencia, C++ es un lenguaje de programación orientado a objetos y C no lo es.

Conceptos básicos de c++ en POO


Clase

Es una abstracción que hacemos de nuestra experiencia sensible, se puede definir en el lenguaje natural como un grupo de cosas con características similares.

Abstracción:

Proceso mental en el que se extraen características de algo en especifico, ignorando detalles que no son de utilidad, es decir, es el pensar y sacar características de un objeto, que se considere que son las necesarias.

Objetos:

Es un conjunto de atributos y métodos, estos derivan de las clases

• Atributos se puede relacionar con las caracteristicas

• Métodos: Son las acciones

Encapsulamiento:

Tal y como su nombre lo indica, es encerrar o encapsular ciertas características para así no poder manejarlas en el programa.

Ejemplo: Los ordenadores de apple, pues estos vienen sellados de tal manera que no se pueden destapar con la misma facilidad que otros, esto para no tocar sus componentes internos.

Herencia:

Es cuando una clase nueva, se crea a partir de una clase existente, heredando atributos y métodos.

Polimorfismo:

Si se tienen los objetos: Persona, Perro, Vaca y a estos se les manda la orden de comer, a pesar de ser la misma orden, cada uno come de una manera distinta.
