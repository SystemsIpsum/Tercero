Interfaces gráficas


Las interfaces gráficas pueden contener

• Labels:

Son especiales para insertar texto en el JFrame.

• TextField:

Son necesarios para que el usuario pueda ingresar datos pedidos o cualquier tipo de dato que se necesite en otro procedimiento.

• Button:

Son los cuales dan acción a un procedimiento establecido en código anteriormente.

• CheckBox:

Da diferentes opciones las cuales solo se podrá seleccionar uno.

• List:

La opción de list nos permite almacenar datos en columna los cuales se pueden combinar con TextField y button para que de acción a guardar

• Table:

La opción table nos deja almacenar datos por filas y columnas las cuales podremos guardar en un array, arraylist o ingresarlos con un vector.

Funcionamiento de array y arraylist


Array:

En un array se puede almacenar datos establecidos hasta un determinado limite el cual se establece en el código del array.

• Para llenar un array se tendrá que crear primero con que datos se llenara y sus posiciones.

• Después crearemos un for para ingresar los datos

• Y si queremos imprimir el array, crearemos otro for pero ahora con un soutprint

ArrayList:

Es un array pero indefinido el cual tomara todos los datos que le dijiste e ira creando los campos necesarios para el funcionamiento del mismo.

• Para llenar un arraylist crearemos un constructor con los datos que se quieran ingresar y generaremos los geters de cada dato.

• Después se creará un ArrayList e ingresamos los datos que creamos en el constructor y si se requiere podremos crear un try catch para cualquier error que se genere.

• Para imprimir un arraylist en una tabla necesitaremos en código fuente de la tabla e ingresaremos como dato la matriz que ya tenemos
